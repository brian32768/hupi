Pour votre information.

Mon analyse est que des Réseaux Express Vélo permettraient, entre autre, le
développement des VPH les plus efficaces (vélos-couchés, vélomobiles...).
Les télécabines dont il est question devraient être intéressantes dès que le
terrain est difficile : eau, relief, forte densité urbaine (je partage donc
le point de vue de Jean-Luc Saladin). Utilisées avec parcimonie, elles
devraient pouvoir s'intégrer sans trop de difficulté dans le paysage urbain.
Voici le lien :
http://www.mediaseine.fr/Les-Mureaux-La-municipalite-02071

Une carte pour mieux comprendre les enjeux des Mureaux ;
