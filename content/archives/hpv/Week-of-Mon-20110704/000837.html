People always start slowly backing away from me when I start talking  
like this ... but...

I recall a planetary BB gear system that gave you one gear when  
pedaling forward, and another when pedaling backward.  I believe there  
is a chain loop system that does the same thing. I don't care so much  
about the gear change, but it always seemed to me that being able to  
change the way one is using ones legs might provide some welcome  
variety on long rides.  On a conventional bike you can move about a  
lot more, including standing on the pedals.  Not so much on a 'bent.
